# SIEM-Based-Fraud-Analytics

## Overview
This project focuses on detecting suspicious financial transactions using rule-based anomaly detection and dashboard-based fraud monitoring.

## Features
- Fraud detection rules
- Risk scoring engine
- Suspicious transaction monitoring
- Dashboard visualization
- SIEM-style alert simulation

## Technologies Used
- Python
- Pandas
- Google Colab
- Power BI

## Fraud Detection Logic
- High-value transfer detection
- Account-draining behaviour analysis
- Cash-out monitoring
- Risk threshold scoring

## Dashboard Features
- Fraud distribution visualization
- Transaction category analysis
- High-risk activity monitoring
